# Roadmap v1.0.0 개요

이 문서는 `docs/milestones/v1.0.0.md`를 기준으로 v1.0.0 개발 카테고리를 미리 분류한 문서입니다.

## 카테고리

1. Foundation
2. Runtime Loader
3. React Core API
4. Overlay API
5. Events, Types, Docs
6. Release

## 운영 규칙

- 모든 구현/문서/테스트 작업은 위 카테고리 중 하나에 먼저 매핑합니다.
- 이슈/PR 생성 시 카테고리와 phase를 동시에 표기합니다.
- `phase:1` 완료 전 `phase:2+` 기능 merge를 금지합니다.

## 권장 라벨

- 도메인 라벨: `infra`, `feature`, `docs`, `test`, `breaking`, `submodule`
- 단계 라벨: `phase:1` ~ `phase:6`
