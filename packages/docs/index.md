# react-naver-maps-kit

네이버 지도 SDK를 React에서 안전하게 다루기 위한 모던 툴킷입니다.

## 포함 기능

- `NaverMapProvider`: SDK 로딩 상태, 에러, 지도 인스턴스를 컨텍스트로 관리
- `NaverMap`: 지도 생성/업데이트/정리 생명주기 자동 처리
- `useNaverMap`, `useNaverMapInstance`: Provider 안전 가드가 포함된 훅
- `loadNaverMapsScript`: 독립 실행 가능한 SDK 로더 유틸
